Privacy Policy

Smart Clipboard Manager is a Chrome Extension that prioritizes user privacy.

What Data We Collect

This extension does not collect, transmit, or share any personal data. All clipboard history and settings are stored locally in your browser using chrome.storage.local.

Clipboard Access

Clipboard content is accessed only when the user interacts with the extension or copies text from a page. No automatic or hidden access occurs.

Data Sharing

No clipboard data is sent to external servers. The extension does not use analytics, cookies, or third-party tracking tools.

Permissions

The extension requests only the permissions necessary for clipboard tracking and storage. These are used solely to deliver the advertised functionality.
